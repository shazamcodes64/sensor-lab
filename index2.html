SENSOR REFERENCE
PROXIMITY SENSOR
Detects objects without contact via EM field or radiation. Best for: distance detection, presence detection, automatic triggers.
IR SENSOR
Emits/detects infrared radiation. Best for: remote control signals, object detection via reflection, motion/heat detection.
LCD OUTPUT
Liquid crystal display. Best for: showing data, status, images. Requires a signal input — cannot detect anything itself.
