SENSOR REFERENCE
LCD — LIQUID CRYSTAL DISPLAY
Output device only. Uses liquid crystals between polarizing filters. Voltage OFF → crystals twist → light passes → bright. Voltage ON → crystals untwist → light blocked → dark. Cannot detect anything.
PROXIMITY SENSOR
Detects objects without contact. Types: Inductive (metal, eddy currents), Capacitive (any material), Magnetic (reed contacts), Photoelectric (light beam). Used in phones, doors, parking, faucets.
IR SENSOR
Active IR: LED emitter + photodiode receiver. Detects by reflection. Passive IR (PIR): detects body heat. Used in remotes (38 kHz modulation), burglar alarms, thermometers, robots.
